<P><A href="http://www.paypal.com/"><IMG alt=PayPal hspace=5 src="http://blog.xdanger.com/archives/pics/20040115a.gif" align=left vspace=5 border=0></A>　　<A href="http://www.paypal.com/">PayPal</A>是全球最大的在线付款平台，使用网络刷卡方式，支持几乎所有国际通行信用卡<IMG alt="" hspace=0 src="http://blog.xdanger.com/archives/pics/20040115f.gif" align=absMiddle border=0>，实现即时付款，是网上跨国交易最普遍的形式。同时又是收取服务费用最少的一个。</P>
<P>　　要使用PayPal实现信用卡在线消费，必须注册一个PayPal帐号，并且验证你的信用卡。</P>
<H3>注册帐号步骤：</H3>
<OL>
<LI>登陆PayPal主页：<A href="http://www.paypal.com">http://www.paypal.com</A> 
<LI>点击右上角<A href="https://www.paypal.com/cgi-bin/webscr?cmd=_registration-run">Sign Up进入注册页</A>。 
<LI>选择Personal Account in:<STRONG>China</STRONG>，然后Continue。 
<LI>然后就是常常看到的一些表单，出了一个Work Telephone之外都是必填项。 
<LI>在你所填的邮箱中查收验证邮件，点击其中的一个验证连接激活你的邮件地址。PayPal虽然使用电邮地址登陆，但却支持邦定多个邮件地址，每注册一个都需要验证。</LI></OL>
<P>　　这样就完成了帐号注册，不过要使用信用卡消费，还必须注册你的信用卡。</P>
<br /><br />
<H3>信用卡注册步骤：</H3>
<OL>
<LI>在<A href="https://www.paypal.com/row/cgi-bin/webscr?cmd=_account">My Account</A> -&gt; <A href="https://www.paypal.com/row/cgi-bin/webscr?cmd=_profile-summary">Profile</A> -&gt; <A href="https://www.paypal.com/row/cgi-bin/webscr?cmd=_profile-credit-card-new">Credit Cards</A>&nbsp;-&gt; <A href="https://www.paypal.com/row/cgi-bin/webscr?__track=_profile-credit-card-new:p/acc/pro/cc:_profile-credit-card-new-submit">Add</A>填写信用卡资料。Card Verification Number 是信用卡背面的最后三个数字<IMG alt="Credit Card Verification Number" hspace=0 src="http://blog.xdanger.com/archives/pics/20040115b.gif" align=absMiddle border=0>。<STRONG>配合信用卡的卡号，任何人都可以从网上刷走你的钱。</STRONG>所以一定要在受信赖的网站才能填写这些资料。并且容易被我们忽视的是，<STRONG>一定要注意信用卡卡号的保密</STRONG>，这个就像CD-Key或手机冲值卡的密码一样，几乎起到证码的作用，而不只是一个编号。 
<LI>注册信用卡之后，PayPal会向您要求从你的卡中扣除$1.95，这个是用来验证你的信用卡的确可以透支并适合在网上使用。不用担心这$1.95，因为在全部注册完成之后PayPal会将它退还给你。这步确认就可以了。 
<LI>如果你的信用卡颁发机构支持在线查询的话，过1-3个工作日，你就可以在网上查到一笔美元刷卡交易记录，如下图所示（因为我注册了2张卡，所以有2条记录）：<A onclick="window.open('http://blog.xdanger.com/archives/pics/20040115c.html','popup','width=727,height=209,scrollbars=no,resizable=no,toolbar=no,directories=no,location=no,menubar=no,status=no,left=0,top=0'); return false" href="http://blog.xdanger.com/archives/pics/20040115c.html"><IMG style="MARGIN: 10px" height=146 alt="4-digit Expanded Use Number" src="http://blog.xdanger.com/archives/pics/20040115g.jpg" width=509 align=left border=0></A><BR clear=all>圈出来的就是交易验证码，在<A href="https://www.paypal.com/row/cgi-bin/webscr?cmd=_account">My Account</A> -&gt; <A href="https://www.paypal.com/row/cgi-bin/webscr?cmd=_profile-summary">Profile</A> -&gt; <A href="https://www.paypal.com/row/cgi-bin/webscr?cmd=_profile-credit-card-new">Credit Cards</A>中选取信用卡然后点Complete Expanded Use Enrollment：<IMG style="WIDTH: 600px; HEIGHT: 190px" alt="" hspace=10 src="http://blog.xdanger.com/archives/pics/20040115d.png" align=baseline vspace=10 border=0><BR clear=all>然后填入那4位注册码，提交后就OK了：<IMG style="WIDTH: 600px; HEIGHT: 98px" alt="" hspace=10 src="http://blog.xdanger.com/archives/pics/20040115e.png" align=baseline vspace=10 border=0><BR clear=all>
<LI>PayPal支持每个帐户拥有多张信用卡，申请步骤一致。</LI></OL>
<P>　　这样就算基本完成了。可以轻松实现网上付款，比如在<A href="http://www.ebay.com/">eBay</A>购物，而不必再为如何跨国付款及人民币-美金结算问题伤脑筋了（前提是信用卡颁发机构具有美金消费人民币还款业务），因为多数网上商家都支持PayPal支付。你还可以和远在异国的亲朋好友寄款/收款。</P>
<P>　　唯一遗憾的是，PayPal只支持少数几家美国银行的转帐汇款。对于我们来说，PayPal只能消费而无法将收到的钱提现，收到的钱是放在你在PayPal的帐户里的，而并非直接转入信用卡，要么你在美国有熟人，否则这些钱最好的办法就是在网上消费掉。因为信用卡消费是有国际惯例的。比如你是<A href="http://www.visa.com/">Visa</A>卡，就可以在全世界有Visa标志的POS机上刷卡或ATM机上提现，因为这些发Visa卡的机构都和Visa有清算协议，PayPal只是虚拟化了这一刷卡过程。同理，你不可能跑到全世界任意一个颁发Visa卡的机构去向你的信用卡存钱，所以PayPal上也不行。</P>
