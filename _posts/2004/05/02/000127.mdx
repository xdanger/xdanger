这个月的确有点喘不过气来。先是 Java 考试；然后前后断断续续做了一年的[ExSun.net](http://exsun.xdanger.com/)最后还是放弃了；然后是忙着逛太平洋买[笔记本](http://www.asus.com.cn/products/notebook/l4000r/overview.asp?model=L4000R%u7CFB%u5217)；砍了一个星期的价买回来之后发现兜里没钱了，又拼了一星期做了个[站](http://fastwell.xdanger.com)换生活费（帐还没结给我……555），其间好好研究了下[PEAR](http://pear.php.net)和 MVC；这个星期在啃[JSP](http://java.sun.com/products/jsp/)，发现开发 Java 项目爆复杂……

先算是做个总结吧，马上分批写点东西下来。
