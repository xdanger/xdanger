从 [2003-12-04 开始用 MovableType][MovableType]，换了 N 台服务器，期间数据库出过一次问题，无法跟着升级，重新装了一遍从头开始。到 09 年 twitter 起来后，就实在懒得在博客再发东西了。一晃好多年过去了，一直觉得有必要再重新打理下主页，觉得 MT 麻烦，又不想用 WP 那么俗，也不想再折腾 PHP 了，没啥思路。

直到前段时间看到 GitHub Pages 用 Jekyll Host 静态网站，觉得这想法太酷了，就是我想要的最简单的一个很简单可维护的静态 CMS 系统。于是双休日颓废了下，折腾了 2 天把整个环境弄好了，生成的静态页同时 Push 到又拍云和 GitHub，所有历史记录都有 Git 永久保存，全部是文本文件，非常有安全感 ~

其中花了 1 天重新整了个风格，嘿嘿。
