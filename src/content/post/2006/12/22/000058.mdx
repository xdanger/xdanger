Google Map 太好用了，确切地说，[他的 API](http://www.google.com/apis/maps/) 太好用了。现在好的地图服务实在太多了，但是花点时间在产品对外的 API 上，是有事半功倍的效果的。一个好的 API 设计，齐全的文档，可以省无数市场推广费用，还有无法估量的无形资产效应。像这个，想用 Baidu 地图做都没办法呢……

仓促做的，好像挪到很多地方时间有 Bug……看看丘健同学现在在的[法国小城](http://www.xdanger.com/vendors/sun-time/45.43785146,4.39315796)，竟然黑夜有 15 个小时多，可怜啊。
