description: "本文揭露了 2006 年 VeryCD 网站遭遇的恶意网页篡改事件。作者发现网站使用的网宿 CDN 服务器在用户访问 lib.verycd.com 时，私自修改了原始 HTML 代码，插入弹窗广告代码，导致用户不断收到指向 link789.com 的广告弹窗。文章提供了用户反馈、聊天记录和 FlashGet 截图等证据，并列出了被确认有问题的网宿服务器 IP 地址。这一事件揭示了当时中国互联网基础设施提供商可能存在的不当行为，体现了早期中国互联网发展中的诚信问题。"

最近 VeryCD 论坛里很多用户问到（主题[1](http://board.verycd.com/t286676.html)、[2](http://board.verycd.com/t294103.html)、[3](http://board.verycd.com/t286133.html)、[4](http://board.verycd.com/t294602.html)）为什么打开 lib.verycd.com 就会不断弹广告窗口出来，网址为：<http://www.link789.com/kuho.html。起初一直以为有流氓软件估计搞我们，今天无意中发现，原来是我们所使用的[网宿>](<http://www.chinanetcenter.com/>) CDN 服务器篡改了我们原本页面的 HTML 代码。

打电话给网宿的人，他们看过后告诉我们没有问题，而此时确实恢复了正常。还好打电话之前有好几位网友为我们截了图，还有[陈少举](http://www.vnnfans.org/)同学干净利落地录下了证据：

1. 我和陈少举对话时的即时录像 - .swf 文件 1.38MB - [察看](http://files.xdanger.com/verycd/cnc.popupwindow.record1.swf)；
2. 陈少举重现网通用户访问 lib.verycd.com 时的弹窗录像 - .swf 文件 373KB - [察看](http://files.xdanger.com/verycd/cnc.popupwindow.record2.swf)。
